Senior Housing Management System

This project is a systems analysis and design of Senior Housing Management System project for the Beijing Hongfu Hotel Group. The system is a web-based system designed with HTML5 and Open Source technologies like PHP, MySQL. The system front-end is an e-commerce like Web pages designed with HTML5, JQuery and PHP. The system back-end is a senior housing system database designed with MySQL. I have implemented WebRTC and sipML5 with OpenIMS for audio and video communication.
